# Imágenes del Proyecto

## Imagen de Fondo Login

**Archivo necesario:** `concrete_background.jpg`

**Ubicación:** Coloca tu imagen de concreto aquí con el nombre exacto:
```
assets/images/concrete_background.jpg
```

**Especificaciones recomendadas:**
- Formato: JPG o PNG
- Resolución: 1080x1920px o superior
- Peso: Máximo 500KB (para rendimiento)
- Estilo: Textura de concreto/hormigón prefabricado
- Color: Gris concreto natural

**Tipos de imagen que funcionan bien:**
- Textura de concreto pulido
- Superficie de prefabricado
- Muro de hormigón
- Bloques de concreto

La imagen se mostrará con una capa oscura encima (opacity 0.7) para que las letras blancas sean legibles.

## Cómo Agregar las Imágenes

1. Copia tu imagen de concreto a esta carpeta
2. Renómbrala a: `concrete_background.jpg`
3. La app automáticamente la detectará y usará

**Importante:** El nombre debe ser exactamente `concrete_background.jpg`
